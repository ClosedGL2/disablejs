Using an event listener instead of href="" for links

This isn't common, but apparently someone thought this was a good idea, and this is a real screenshot.

This link uses JavaScript to open the link's destination in a new tab when the link is clicked. When someone right clicks the link and clicks "Open Link in New Tab", the browser will just see the href="" and go to the site's home page.

To make a link open in a new tab, you just need to add target="_blank", and you'll have a functional link instead of whatever this is.
